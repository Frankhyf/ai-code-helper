# AI应用生成平台 - 设计风格指南

## 设计理念

### 视觉语言
- **现代简约**: 采用干净的线条和充足的留白，突出内容本身
- **科技感**: 使用渐变、发光效果和动态元素体现AI技术特性
- **专业可信**: 通过一致的配色和精致的细节建立用户信任

### 色彩方案
- **主色调**: 深蓝渐变 (#1e3a8a → #3b82f6) - 体现技术专业性
- **辅助色**: 青绿色 (#06b6d4) - 用于强调和交互元素
- **中性色**: 灰色系 (#f8fafc, #64748b, #334155) - 背景和文本
- **功能色**: 
  - 成功: #10b981
  - 警告: #f59e0b  
  - 错误: #ef4444

### 字体系统
- **标题字体**: Inter (粗体) - 现代感强，易读性好
- **正文字体**: Inter (常规) - 保持一致性
- **代码字体**: JetBrains Mono - 等宽字体，适合代码显示

## 视觉效果

### 核心库应用
- **Anime.js**: 页面切换动画、按钮交互反馈、数据加载动画
- **ECharts.js**: 应用统计图表、用户数据分析可视化
- **Pixi.js**: Hero区域粒子效果、背景动态元素
- **Matter.js**: 应用卡片物理动画、拖拽交互效果
- **p5.js**: 创意编码背景、AI生成过程可视化

### 动画效果
- **页面加载**: 渐进式内容显示，从上到下依次展现
- **交互动画**: 按钮悬停3D倾斜效果，卡片翻转动画
- **状态转换**: 平滑的过渡动画，避免突兀的变化
- **数据可视化**: 图表动画展示，数值变化时的动态效果

### Hero区域设计
- **背景**: 使用p5.js创建的动态粒子网络效果
- **布局**: 左侧标题和描述，右侧输入区域
- **特效**: 粒子连线动画，鼠标交互响应
- **渐变**: 从深蓝到浅蓝的径向渐变叠加

### 组件样式
- **卡片设计**: 圆角矩形，轻微阴影，悬停时抬升效果
- **按钮样式**: 渐变背景，圆角设计，点击时缩放反馈
- **表单元素**: 简洁边框，聚焦时发光效果
- **导航栏**: 半透明背景，毛玻璃效果

### 响应式适配
- **桌面端**: 充分利用屏幕空间，多列布局
- **平板端**: 适中的间距和字体大小，保持可用性
- **移动端**: 单列布局，大按钮设计，触摸友好

### 微交互设计
- **加载状态**: 骨架屏和进度指示器
- **反馈提示**: Toast消息和模态框
- **状态指示**: 图标和颜色变化表示不同状态
- **手势支持**: 滑动、长按等移动端手势
